# UAE Migration Nowcasting Model — Overview

## How the Model Works

This study uses a **Bayesian Hierarchical Model** to estimate the number of international migrants living in the UAE by combining four imperfect data sources — FCSC administrative records, Facebook "Lived In" data, Facebook "Recently In" data, and the UAE Labour Force Survey — each of which captures a different, biased slice of the true migrant population. At its core, the model treats each country-to-UAE migration corridor as an **autoregressive time series (AR1)** in log-space, where the true (unobserved) migrant stock for each corridor gradually evolves over time while reverting towards a corridor-specific mean level. A hierarchical structure allows corridors to "borrow strength" from one another — so even corridors with sparse data benefit from patterns observed in data-rich corridors like India or Pakistan. For each data source, the model simultaneously estimates a **bias correction factor (γ)**: for example, it learns that admin records capture roughly 76% of the true stock (γ ≈ 0.76), while Facebook MAU reflects about 51% (γ ≈ 0.51). By jointly estimating these biases alongside the latent true stocks, the model produces unified migration estimates with full uncertainty quantification — something no single data source can provide alone.

## Significance of the Output

The model's output demonstrates that integrating multiple imperfect data sources through a principled Bayesian framework produces **coherent, realistic migration estimates** across all 25 origin corridors, with India estimated at ~4,500K, Pakistan at ~1,760K, and total UAE expat population at ~10,900K — figures consistent with known demographic patterns. Critically, the model provides **80% credible intervals** for every estimate, enabling policymakers to quantify uncertainty rather than rely on point estimates from a single registry. The estimated bias parameters reveal that each data source systematically undercounts the true population to varying degrees, validating the core hypothesis that no single source is sufficient. For policy applications, these estimates can directly inform healthcare capacity planning, labour market visa quotas, and urban infrastructure investment across the UAE. While these results are based on simulated data as a methodological proof-of-concept, the framework is designed to be directly applicable to real Facebook Marketing API and FCSC administrative data for operational migration nowcasting.
